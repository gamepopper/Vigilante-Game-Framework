Applys a set of functions to a specified State ID in the FSM.

Parameters
stateIDThe ID number of the state you wish to apply the functions to.
onUpdateFuncThe function that gets called for each Update call. Needs to be a function that returns a signed integer (the next state ID) and passes in a VBase class and float (delta time) as a parameter. Required for the FSM to operate.
onEnterFuncThe function that gets called when the FSM first enters a state. Needs to be a function that passes in a VBase object as a parameter. (optional)
onExitFuncThe function that gets called when the FSM exits a state. Needs to be a function that passes in a VBase object as a parameter. (optional)
